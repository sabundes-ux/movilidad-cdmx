Pensamiento Computacional

  • Descomposición: El proyecto se divide en varias partes, incluyendo la recopilación de datos, el análisis de movilidad, la visualización de resultados y la documentación.
  • Reconocimiento de patrones: Se identifican patrones en los datos de movilidad para comprender mejor las tendencias y comportamientos de los usuarios.
  • Abstracción: Se simplifican los datos complejos para crear visualizaciones claras y comprensibles.
  • Algoritmos: Se utilizan algoritmos para procesar y analizar los datos de movilidad, así como para generar visualizaciones efectivas.
